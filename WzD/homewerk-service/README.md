# HomeWerk Service public CI mirror

This folder is reserved for cleaned source code copied from the private WzD / HomeWerk Service repository.

Public mirror rules:
- source code only;
- no real customer data;
- no PDFs, images, scans, invoices or receipts;
- no `.env`, secrets, tokens or private URLs;
- no private stage handoff reports.

The first mandatory check is `tools/public_safety_check.py`.
